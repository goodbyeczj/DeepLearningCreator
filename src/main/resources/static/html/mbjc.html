背景:

睿客实时目标检测器是针对特定领域的实时目标检测任务(ImageTracker ImageTracker)，主要目的是在一段视频影像中识别出备选集中存在的某个目标，是经典的计算机视觉任务。该项目落地形式为在用户翻阅画册过程中,识别出画册所属页数。

数据:

输入数据：输入为实时的视频数据，ImageTracker 的实现是基于经典 SIFT 算子图像特征点发现算法的改良形式 SURF 算法，通过Opencv 和 python 多线程操作，实现了调多线程操作，实现了调用摄像头同步展示视频流与目标检测任务。 用摄像头同步展示视频流与目标检测任务。SURF 是一种兴趣点检测及描述子算法。可以说是对 SIFT 算法的加强版，是用来实现尺度不变性特征点的检测与匹配任务的。SURF 算法首先利用 Hessian 矩阵确定候选点，然后进行 NMS(非极大值抑制)初步确定特征点，再通过确定特征点主方向，综合构造 SURF 特征点描述算法从而进行图像特征抽取。 输出数据：用户翻阅画册过程中,识别出画册所属页数。
